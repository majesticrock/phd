We compute chi(omega, q) for select omega.
We can plot this susceptibility along high-symmetry lines.

Additonally, we compute V_screen(q, omega) = V(q) / (1 + chi(omega, q)) and the average
U = rho_F sum_q V_screen(q, omega) / (N^3)
with N=32 and q from the first BZ.
For realisitc parameters, this yields U~0.4 or U~0.5.
